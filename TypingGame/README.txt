CIS542 Android Application: Typing Game
Name: Yayang Tian

Introduction:
Android typing game in which the user is shown a sentence and has to type it in word-for-word. When the user is done, the program indicates how long it took to type the sentence, assuming it was entered correctly. The application also keeps track of the best (lowest) time to type the sentence correctly.

Components:
Android emulator
Android GUI elements
Handling user interaction, e.g. button clicks
Persistent storage: SQLite

Environment:
SDK and Eclipse are used in ENIAC Machine
