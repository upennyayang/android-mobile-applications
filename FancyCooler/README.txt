CIS542 Final Project:  Fancy Cooler/ Coffee Cooler
Spring 2013
Name: Yayang Tian

Introduction:
Implement an Android application that could cool coffee down, including a front-end Android user interface in Java and a backend C server, SQLite database, Arduino chip with fans,temperature, proximity, and sound sensors.

Components:
Android GUI and socket connection to Andriod
Persistent storage: SQLite
C sever and serial connection to Arduino chip
Arduino low-level control

Environment:
SDK and Eclipse are used in ENIAC Machine
